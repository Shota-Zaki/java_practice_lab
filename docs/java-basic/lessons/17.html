コンパイルエラーと実行時例外

コンパイルエラーはプログラムをclassファイルに変換できない状態です。実行時例外はコンパイルは通るが、実行中に問題が起きる状態です。

int x;
System.out.println(x); // コンパイルエラー
String s = null;
System.out.println(s.length()); // 実行時例外
