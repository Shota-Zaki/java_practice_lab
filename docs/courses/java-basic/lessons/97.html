Lesson 97 入力値をどう扱うか

プログラムの外から来る値は、必ず文字列や不正な値の可能性を考えます。

コマンドライン引数

args は String[] です。数値として使うなら Integer.parseInt などで変換します。

int count = Integer.parseInt(args[0]);

失敗する例

int count = args[0]; // Stringをintに直接代入できない

注意

変換に失敗すると実行時例外になります。Silverでは「コンパイルは通るが実行時に失敗する」問題として出やすいです。
