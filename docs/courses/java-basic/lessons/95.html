Lesson 95 基礎解説の総まとめ

ここまでで、Javaを読むための土台を一通り見ました。次は目的別に進む段階です。

Bronzeへ進む人

文法の基本、クラス、オブジェクト、配列、条件分岐、ループを固めます。暗記よりも短いコードを動かすことが重要です。

Silverへ進む人

コンパイル可否、実行時例外、出力結果、参照比較、継承、例外処理を重点的に練習します。

Goldへ進む人

ラムダ、Stream、ジェネリクス、I/O、並行処理、モジュールを体系的に学びます。基礎が曖昧なままGoldへ行くと詰まります。
