ターミナル・javac・javaコマンド

Javaを実行する最小単位は、ソースファイルをコンパイルしてクラスを実行する流れです。IDEを使う前にコマンドの意味を押さえます。

基本の流れ

Javaのソースファイルは.javaです。javacでコンパイルすると.classができます。javaコマンドはクラス名を指定して実行します。

javac Sample.java
java Sample

よくある間違い

java Sample.class と書いてしまうのは誤りです。javaコマンドにはファイル名ではなくクラス名を渡します。

// NG
java Sample.class

// OK
java Sample

Java 11以降の単一ファイル実行

Java 11以降では、単純なプログラムなら java Sample.java のように直接実行できます。ただし、試験問題ではコンパイルと実行の前提をよく読みます。
