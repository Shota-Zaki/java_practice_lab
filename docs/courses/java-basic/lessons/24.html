Java基礎 Lesson 24

ターミナル・javac・java

手元で動かす基本。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

ターミナル

コマンドを文字で入力してPCへ指示する画面。WindowsならPowerShellやコマンドプロンプトが該当する。

カレントディレクトリ

今いるフォルダ。javacやjavaは基本的にこの場所を基準にファイルやクラスを探す。

基本コマンド

cd 作業フォルダ
javac Sample.java
java Sample

罠

ファイルが見つからない、クラスが見つからない、mainが見つからない、はそれぞれ原因が違う。エラーメッセージを読み分ける。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
