Lesson 81 mainメソッドの形を正確に読む

mainメソッドはJavaプログラムの起点です。似た形の間違いが多いので、形で覚えるだけでは危険です。

基本形よくある誤り起動引数

基本形

public static void main(String[] args) { }

String... argsも可変長引数として有効です。

よくある誤り

誤り理由
void main(String[] args)起動用mainとしてはpublic staticが必要
public void main(...)staticがない
public static int main(...)戻り値がvoidではない

起動引数

java Sample A Bで実行すると、args[0]は"A"、args[1]は"B"です。
