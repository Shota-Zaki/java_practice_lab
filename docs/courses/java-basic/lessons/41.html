メソッド呼び出しと引数

メソッドへ値を渡す仕組みを理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

Javaは値渡し

Javaのメソッド引数は値渡しです。参照型の場合も、参照のコピーが渡されます。

static void change(StringBuilder sb) {
    sb.append("B");
}
StringBuilder s = new StringBuilder("A");
change(s);
System.out.println(s); // AB

参照先の変更と参照変数の付け替え

メソッド内でオブジェクトの中身を変えると呼び出し元にも見えます。しかし、引数変数そのものを別のオブジェクトへ代入しても、呼び出し元の参照は変わりません。

static void reset(StringBuilder sb) {
    sb = new StringBuilder("X");
}
StringBuilder s = new StringBuilder("A");
reset(s);
System.out.println(s); // A

試験での注意

「参照のコピー」がわかっていないと、メソッド呼び出し後の出力問題で落ちます。
