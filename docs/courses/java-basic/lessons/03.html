ソースコードから実行まで

Javaの基本の流れは、ソースコード作成、コンパイル、実行です。

javac Sample.java
java Sample

.javaファイル

人間が書くファイルです。public classがある場合、基本的にファイル名はpublic class名と一致させます。

.classファイル

`javac`が生成するバイトコードです。JVMがこれを読み込んで実行します。

よくある間違い

`java Sample.java`は単一ファイルソースコード実行として使える場合もありますが、通常の学習では`javac`でコンパイルして`java クラス名`で実行する流れを押さえます。
