OS・JDK・JRE・JVM

Javaを動かすには、OSの上でJava実行環境が動きます。開発にはJDK、実行にはJVMが関係します。

OS

Windows、macOS、LinuxなどがOSです。ファイル、メモリ、プロセス、ネットワークなどを管理します。

JDK

JDKはJava Development Kitです。`javac`、`java`、標準ライブラリ、開発用ツールを含みます。コードを書く人は基本的にJDKを使います。

JVM

JVMはJava Virtual Machineです。コンパイルされた`.class`ファイルのバイトコードを解釈・実行します。

試験での注意

`javac Sample.java`でコンパイルし、`java Sample`で実行します。`java Sample.class`ではありません。
