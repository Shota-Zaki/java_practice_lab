標準入力・標準出力・標準エラー

プログラムとターミナルの入出力を理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

標準出力

System.out.printlnは標準出力へ文字を出します。

標準エラー

例外スタックトレースなどは標準エラーへ出ることがあります。学習初期は、出力結果とエラー表示を区別しましょう。

試験での注意

Java Silverでは標準入力より、出力結果・例外・コンパイル可否が中心です。
