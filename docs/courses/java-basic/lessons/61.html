パスとカレントディレクトリ

コマンドでJavaを動かすときに必要なパスの考え方を整理します。

カレントディレクトリとは現在地

ターミナルには現在の作業場所があります。javac Main.javaは、基本的に現在地からMain.javaを探します。

相対パスと絶対パス

相対パスは現在地からの位置、絶対パスはドライブやルートからの完全な位置です。

Java学習での使いどころ

packageを使うとフォルダ構成が重要になります。package com.exampleなら、通常はcom/example配下にclassが配置されます。

小さな例

cd project
javac src/Main.java

確認ポイント

  • ターミナルには現在地がある
  • ファイルが見つからない原因はパス違いが多い
  • packageとフォルダ構成は関連する
