デバッグの基本

デバッグでは、エラーメッセージ、行番号、変数の値、実行順を確認します。Java Silverの問題でも同じで、焦って出力だけ追うとコンパイルエラーを見落とします。

まずコードが成立しているか、次に例外が出ないか、最後に出力を追うのが実務でも有効です。
