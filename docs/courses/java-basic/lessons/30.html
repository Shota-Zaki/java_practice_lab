標準ライブラリの読み方

Javaでは自分で全てを書くのではなく、標準ライブラリのクラスとメソッドを使います。

APIとは

APIは、プログラムから利用できる機能の入口です。String、ArrayList、Arrays、Integerなどは試験でも頻出です。

メソッドを見る観点

メソッドは、名前、引数、戻り値、変更の有無、例外の有無で見ます。特にStringは不変なので、replaceやsubstringは元の文字列を変えません。

String s = "abc";
s.replace('a', 'x');
System.out.println(s); // abc

s = s.replace('a', 'x');
System.out.println(s); // xbc

試験での注意

メソッド名を知っているだけでは不十分です。戻り値を受け取る必要があるか、元のオブジェクトを変更するか、範囲外で例外が出るかまで確認します。
