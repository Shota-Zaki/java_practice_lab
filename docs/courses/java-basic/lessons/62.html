標準ライブラリと外部ライブラリ

Javaに最初から入っている機能と、後から追加する機能を分けます。

標準ライブラリ

String、ArrayList、HashMap、LocalDateなどはJava標準ライブラリに含まれます。importが必要なものもあります。

外部ライブラリ

JUnit、Spring、Gsonなどは後から追加して使うライブラリです。MavenやGradleで管理することが多いです。

試験では標準ライブラリ中心

Java Silverでは標準APIの基本的な使い方が問われます。特にString、StringBuilder、ArrayList、Arrays、Integerあたりは頻出です。

小さな例

import java.util.ArrayList;

var list = new ArrayList<String>();

確認ポイント

  • java.langは自動import
  • java.utilは基本的にimportが必要
  • 試験は標準API中心
