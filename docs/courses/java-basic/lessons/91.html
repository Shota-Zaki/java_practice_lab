Lesson 91 ジェネリクスを怖がらずに読む

ジェネリクスは、コレクションに入る型をコンパイル時に制限する仕組みです。

基本型消去の考え方注意

基本

List<String> list = new ArrayList<>();
list.add("A");
// list.add(10); // コンパイルエラー

型消去の考え方

実行時には型引数の情報は一部失われます。Silverでは深く問われにくいですが、Goldでは重要になります。

注意

List<Object>とList<String>は親子関係ではありません。配列とは違います。
