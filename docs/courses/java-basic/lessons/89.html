Lesson 89 interfaceを読むための最低限

interfaceは「使い方の約束」を定義します。Java Silverではdefaultメソッドや定数も重要です。

interfaceのメソッドinterfaceの変数defaultメソッド競合

interfaceのメソッド

抽象メソッドは暗黙にpublic abstractです。実装クラスでメソッドを実装するときはpublicが必要です。

interfaceの変数

interface内の変数は暗黙にpublic static finalです。つまり定数です。

defaultメソッド競合

複数のinterfaceから同じdefaultメソッドを継承すると、実装クラス側で明示的に解決する必要があります。
