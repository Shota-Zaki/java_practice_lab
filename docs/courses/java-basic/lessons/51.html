nullを怖がらずに理解する

nullの意味とNullPointerExceptionの原因を整理します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

nullとは

nullは「どのオブジェクトも参照していない」ことを表す特別な値です。

String s = null;
// s.length(); // NullPointerException

代入はできるが呼び出しは危険

参照型変数にnullを入れること自体はできます。しかし、その変数からメソッドやフィールドへアクセスすると例外になります。

試験での注意

配列要素の初期値がnullになる問題、String配列でconcatを呼ぶ問題は頻出です。
