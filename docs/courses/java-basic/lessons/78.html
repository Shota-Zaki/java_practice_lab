Lesson 78 2進数・16進数・ビット演算の入口

Javaでは整数リテラルやビット演算が出てきます。Silverでは深いビット演算より、表記と型の読み間違いに注意します。

整数リテラルの表記ビット演算子と論理演算子を混同しない注意

整数リテラルの表記

表記例意味
10進数10通常の数
2進数0b101010
16進数0x0A10
long10Llong型

ビット演算子と論理演算子を混同しない

&や|はbooleanにも整数にも使えます。booleanで使う場合、&&や||と違って短絡評価しません。

int x = 0;
boolean r = false & (++x > 0);
System.out.println(x); // 1

注意

試験では、2進数そのものよりも、短絡評価の有無や、0とOの読み間違い、lと1の読み間違いが事故になります。
