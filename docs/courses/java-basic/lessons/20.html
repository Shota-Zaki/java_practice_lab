Java基礎 Lesson 20

コードを読む順番

初心者向けの読解手順。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

入口

まずクラス定義、main、変数宣言、メソッド呼び出しを分けて見る。

型を見る

変数の宣言型で呼べるメソッドが決まる。代入されている実体型とは分ける。

値の変化

++、+=、メソッド呼び出し、配列要素の変更で値が変わる箇所だけを追う。

出力

System.out.print/printlnの順番を最後に追う。途中で例外が出るなら、その後は実行されない。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
