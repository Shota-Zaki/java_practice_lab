Java学習の進め方

最初は文法を網羅しようとしすぎるより、実行できる小さなコードを増やす方が効率的です。

  1. Javaの実行の流れを知る
  2. 変数・型・文字列を使う
  3. 条件分岐とループを書く
  4. クラスとメソッドを作る
  5. 例外処理を読む
  6. 演習問題で判定力を鍛える
