APIドキュメントの読み方

Java標準APIを読むための最低限の見方を説明します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

見るべき場所

API仕様では、クラスの説明、コンストラクタ、メソッド、戻り値、例外を確認します。

例：String.substring

substring(beginIndex, endIndex)のendIndexは含まれません。

String s = "abcdef";
System.out.println(s.substring(1, 4)); // bcd

試験での注意

戻り値を返すメソッドなのか、元のオブジェクトを変更するメソッドなのかを必ず見分けます。
