Java基礎 Lesson 8

if文と条件式

boolean、比較、短絡評価を読む。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

ifの条件

ifの条件式はbooleanでなければならない。C言語のように0や1を条件として扱えない。

比較演算子

==、!=、<、>、<=、>=を使う。ただし参照型の==は同一オブジェクトかどうかを見る。

短絡評価

&& は左がfalseなら右を評価しない。|| は左がtrueなら右を評価しない。右側に++などの副作用があると結果が変わる。

例

int x = 0;
if (false && ++x > 0) { }
System.out.println(x); // 0

試験での罠

& や | をbooleanに使うと、短絡しない。右側も評価される。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
