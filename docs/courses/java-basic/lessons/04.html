mainメソッドと起動引数

Javaアプリケーションの開始地点はmainメソッドです。

public static void main(String[] args) { }

`String... args`も有効です。起動引数は文字列配列として渡され、クラス名は含まれません。

java Main red blue

この場合、`args[0]`は`red`、`args[1]`は`blue`、`args.length`は2です。
