Java基礎 Lesson 4

mainメソッド

Javaプログラムの入口を正確に読む。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

基本形

public static void main(String[] args) {
    // 実行する処理
}
この形が最も基本。public、static、void、main、String[] args の意味を分けて読む。

staticの意味

mainはインスタンスを作らなくてもJVMから呼び出せる必要があるためstatic。main内からインスタンスフィールドを直接使えないのはこのため。

args

argsはコマンドライン引数を受け取る配列。java Sample A B なら args[0] は A、args[1] は B。引数がないのに args[0] を読むと実行時例外になる。

許される形

String[] args と String... args はどちらも使える。配列名はargsでなくてもよい。ただしmainという名前とstaticなどの条件は重要。

試験での罠

mainの形が少し違ってもコンパイル自体は通ることがある。しかし起動メソッドとして認識されない場合がある。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
