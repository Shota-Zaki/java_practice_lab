Lesson 93 Stream APIの入口

Stream APIは、コレクションなどの要素を流れとして処理する仕組みです。

基本の流れ中間操作と終端操作Goldへの接続

基本の流れ

List.of(1,2,3).stream()
    .filter(n -> n % 2 == 1)
    .map(n -> n * 10)
    .forEach(System.out::println);

中間操作と終端操作

filterやmapは中間操作、forEachやcollectは終端操作です。終端操作が呼ばれるまで処理は実行されません。

Goldへの接続

GoldではStream、ラムダ、Optional、コレクションの理解が大きな比重を持ちます。
