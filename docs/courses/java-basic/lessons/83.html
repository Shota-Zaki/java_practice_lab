Lesson 83 式の途中で値が変わるケース

インクリメントや短絡評価では、式を左から追う力が必要です。

前置と後置短絡評価注意

前置と後置

int n = 1;
System.out.println(n++ + ++n); // 1 + 3

n++は使った後に増えます。++nは増えてから使います。

短絡評価

int x = 0;
if (false && ++x > 0) { }
System.out.println(x); // 0

&&は左がfalseなら右を評価しません。

注意

出力問題では、値の変化をメモしながら追う必要があります。頭の中だけで処理すると高確率でミスします。
