Lesson 85 Stringを読むときの基本姿勢

Stringは不変です。変更系に見えるメソッドでも、元の文字列は変わりません。

戻り値を受け取るか== と equalsintern

戻り値を受け取るか

String s = "abc";
s.replace('a', 'x');
System.out.println(s); // abc

変えたいならs = s.replace('a','x');のように戻り値を代入します。

== と equals

==は参照比較、equals()は内容比較です。ただし、クラスによってequals()の実装は違います。

intern

intern()は文字列プール上の参照を返します。参照比較問題で頻出です。
