可変長引数の入口

String... argsなどの可変長引数を理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

可変長引数とは

String... argsは、メソッド内では配列のように扱えます。

public static void main(String... args) {
    System.out.println(args.length);
}

ルール

可変長引数は仮引数リストの最後に1つだけ指定できます。

void test(int x, String... values) {} // OK
// void bad(String... values, int x) {} // NG

試験での注意

String[] argsとString... argsはmainメソッドとしてどちらも使えます。
