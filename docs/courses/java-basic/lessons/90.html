Lesson 90 recordを初めて使う前に知ること

recordはデータ保持用クラスを簡潔に書くための仕組みです。

自動生成されるものgetterではない注意

自動生成されるもの

recordでは、フィールド、コンストラクタ、アクセサ、equals、hashCode、toStringが自動生成されます。

record Book(String title) { }
Book b = new Book("Java");
System.out.println(b.title());

getterではない

アクセサ名はgetTitle()ではなく、コンポーネント名そのもののtitle()です。

注意

recordのコンポーネントは実質的に再代入できません。状態を後から変える用途には向きません。
