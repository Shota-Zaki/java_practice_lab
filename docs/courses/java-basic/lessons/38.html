値・参照・オブジェクトの超基本

プリミティブ型と参照型の違いを、初心者向けに整理します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

プリミティブ型

intやdoubleなどは、変数に値そのものが入ると考えると理解しやすいです。

int a = 10;
int b = a;
b = 20;
System.out.println(a); // 10

参照型

参照型の変数には、オブジェクトそのものではなく、オブジェクトを指す情報が入ると考えます。

StringBuilder a = new StringBuilder("A");
StringBuilder b = a;
b.append("B");
System.out.println(a); // AB

試験での注意

==は参照が同じかを確認します。中身の文字列やフィールド値が同じかどうかとは別です。
