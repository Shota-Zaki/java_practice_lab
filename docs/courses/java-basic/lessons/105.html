Lesson 105 nullの扱い方

nullは「何も参照していない」状態です。メソッド呼び出しはできません。

失敗例

String s = null;
System.out.println(s.length()); // NullPointerException

安全な確認

if (s != null) {
    System.out.println(s.length());
}

注意

nullはコンパイルエラーではなく、実行時例外につながることが多いです。
