1からわかるJava基礎解説 / Lesson 33

ファイル入出力の入口

Java Goldへ進む前に、ファイルを読む・書く処理の考え方を押さえます。

ファイル入出力とは

プログラムの外にあるファイルからデータを読み込んだり、ファイルへ書き出したりする処理です。例外処理とセットで考えます。

// 入口イメージ。詳細はJava Goldで扱う
// Files.readString(Path.of("sample.txt"));

なぜ例外が必要か

ファイルが存在しない、権限がない、ディスクに書けないなど、外部要因で失敗します。そのためチェック例外やtry-with-resourcesの理解が重要です。

試験での注意

Silverでは深いI/Oは中心ではありません。Goldではjava.nio.file、ストリーム、try-with-resourcesと絡みます。
