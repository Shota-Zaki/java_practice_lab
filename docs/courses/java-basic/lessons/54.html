recordを初めて読む

recordの基本と普通のクラスとの違いを理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

recordとは

recordは、データを保持するクラスを簡潔に定義する仕組みです。

record User(String name, int age) {}

自動生成されるもの

コンポーネントに対応するアクセサ、equals、hashCode、toStringなどが自動生成されます。

User u = new User("A", 20);
System.out.println(u.name());

試験での注意

recordのフィールドは実質的にfinalです。アクセサ名はgetName()ではなくname()です。
