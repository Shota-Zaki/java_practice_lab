変数・型・メモリの考え方

変数は値を入れる箱と説明されがちですが、Javaではプリミティブ型と参照型で考え方が違います。

プリミティブ型

`int`や`double`などは値そのものを扱います。

参照型

`String`や配列、クラスのインスタンスは、オブジェクトそのものではなく参照を変数に持ちます。`==`が参照比較になる理由はここにあります。
