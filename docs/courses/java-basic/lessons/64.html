リテラルの読み方

数値・文字・文字列・nullなどのリテラルを整理します。

リテラルとはコードに直接書いた値

10、3.14、'A'、"Java"、nullなどがリテラルです。

整数リテラル

整数は基本的にintです。longにしたい場合は10LのようにLを付けます。

文字と文字列

'A'はchar、"A"はStringです。見た目が似ていますが型が違います。

小さな例

int a = 10;
long b = 10L;
char c = 'A';
String s = "A";
Object o = null;

確認ポイント

  • 10は基本int
  • 10Lはlong
  • シングルクォートはchar、ダブルクォートはString
