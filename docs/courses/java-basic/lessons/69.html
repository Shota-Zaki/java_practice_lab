メソッドの戻り値とreturn

voidとreturn文の使い分けを整理します。

戻り値があるメソッド

intやStringなどを戻り値型にしたメソッドは、対応する値をreturnする必要があります。

voidメソッド

voidは値を返さないメソッドです。return;で途中終了することはできますが、return 値; はできません。

到達不能コード

returnの後に通常の文を書くと、到達不能コードとしてコンパイルエラーになる場合があります。

小さな例

int add(int a, int b) {
    return a + b;
}

void print(String s) {
    System.out.println(s);
    return;
}

確認ポイント

  • 戻り値型とreturn値を一致させる
  • voidは値を返さない
  • return後の到達不能コードに注意
