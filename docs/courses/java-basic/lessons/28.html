アルゴリズムの入口

Java文法だけでなく、処理の流れを順番に追う力も必要です。

アルゴリズムとは

アルゴリズムは、目的を達成するための手順です。Java Silverでは高度なアルゴリズムは不要ですが、ループ、条件分岐、配列走査の流れを追える必要があります。

int total = 0;
for (int i = 1; i <= 5; i++) {
    total += i;
}
System.out.println(total); // 15

トレース表を作る

出力問題が苦手なら、i、total、条件式の結果を表にして追います。頭だけで追うより、手を動かす方が正確です。

試験での注意

break、continue、return、throwが出ると処理の流れが大きく変わります。ループ問題では、この4つを必ず確認します。
