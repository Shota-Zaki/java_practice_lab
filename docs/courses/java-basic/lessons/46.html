エラーの種類を分類する

コンパイルエラー、実行時例外、論理ミスを分けて理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

コンパイルエラー

文法や型のルールに反しているため、classファイルが作られません。

int x;
System.out.println(x); // ローカル変数未初期化

実行時例外

コンパイルは通るが、実行中に問題が起きるケースです。

String s = null;
System.out.println(s.length()); // NullPointerException

論理ミス

コンパイルも実行もできるが、期待した結果と違うケースです。

int a = 10;
a = a++; // aは10のままになり得る
