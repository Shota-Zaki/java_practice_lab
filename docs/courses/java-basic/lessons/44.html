コレクションの入口

ArrayListとListを初心者向けに整理します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

配列との違い

配列は長さが固定です。ArrayListは要素を追加・削除できます。

ArrayList<String> list = new ArrayList<>();
list.add("A");
list.add("B");
System.out.println(list.size()); // 2

List型で受ける

実装はArrayListでも、変数の型をListにすることがよくあります。

List<String> list = new ArrayList<>();

試験での注意

remove(1)はインデックス削除、remove(Integer.valueOf(1))は要素削除です。
