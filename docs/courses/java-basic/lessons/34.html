テストと確認の基本

プログラムは書いて終わりではなく、意図通りに動くか確認する必要があります。

テストとは

テストは、入力に対して期待した出力や状態になるか確認することです。初心者のうちは、System.out.printlnで途中の値を見るだけでも有効です。

確認すべき観点

正常系だけでなく、境界値、空文字、null、配列の長さ0、範囲外アクセスなどを確認します。

static int doubleValue(int x) {
    return x * 2;
}

System.out.println(doubleValue(3));  // 6
System.out.println(doubleValue(0));  // 0
System.out.println(doubleValue(-2)); // -4

試験での注意

試験では自分で実行できません。だからこそ、普段の学習で実行して確認し、動きの感覚を作っておく必要があります。
