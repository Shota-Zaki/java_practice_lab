配列をメモリ図で理解する

配列変数と配列インスタンスの関係を整理します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

配列もオブジェクト

配列は参照型です。配列変数には配列オブジェクトへの参照が入ります。

int[] a = {1, 2};
int[] b = a;
b[0] = 9;
System.out.println(a[0]); // 9

多次元配列

Javaの多次元配列は「配列の配列」です。行ごとに長さが違っても構いません。

int[][] data = new int[2][];
data[0] = new int[]{1, 2, 3};
data[1] = new int[]{4};

試験での注意

配列のlengthはフィールドです。length()ではありません。
