1からわかるJava基礎解説 / Lesson 35

次に学ぶべき道筋

基礎を終えたあと、Bronze、Silver、Goldへ進むための道筋を整理します。

基礎の到達目標

自分でmainメソッドを書き、変数、条件分岐、ループ、配列、クラス、例外の基本を説明できる状態が最初の目標です。

Bronzeへ進む場合

文法の全体像とオブジェクト指向の入口を固めます。細かいひっかけより、基本語彙と基本構文を優先します。

Silverへ進む場合

コンパイル可否、出力結果、例外発生、参照比較、継承、API仕様を細かく判定する力が必要です。基礎とは別の試験対策が必要になります。

Goldへ進む場合

ラムダ式、Stream API、ジェネリクス、I/O、並行処理、モジュールなど、実務寄りの機能に広がります。Silverの理解が浅いままだとかなり重くなります。
