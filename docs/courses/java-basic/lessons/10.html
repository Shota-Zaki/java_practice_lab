配列とコレクションの入口

配列は固定長、ArrayListは可変長です。配列の長さは`length`、Stringの長さは`length()`です。

int[] a = {1, 2, 3};
System.out.println(a.length);

`ArrayList`は`add`、`get`、`set`、`remove`などを使います。
