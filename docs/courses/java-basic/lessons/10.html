Java基礎 Lesson 10

配列

配列の生成、デフォルト値、参照を理解する。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

配列とは

同じ型の値や参照をまとめて扱う固定長の入れ物。生成後に長さは変えられない。

生成

int[] a = new int[3];
String[] s = new String[3];
int配列の要素は0、String配列の要素はnullで初期化される。

添字

添字は0から始まる。長さ3なら有効な添字は0,1,2。array[3]は実行時例外。

多次元配列

Javaの多次元配列は配列の配列。行ごとに長さが違ってもよい。

試験での罠

配列変数の代入は中身のコピーではなく参照のコピー。片方から要素を変更すると同じ配列を見ている変数にも影響する。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
