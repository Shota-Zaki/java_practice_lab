JDKを入れると何ができるか

JDK、javac、java、jarの役割を整理します。

JDKは開発用の道具セット

JDKにはJavaプログラムを作るためのツールが含まれます。代表的なのがjavacとjavaです。

javacとjavaの違い

javacはコンパイル、javaは実行です。初心者はこの2つを混同しがちですが、役割は明確に違います。

jarはまとめる道具

jarは複数のclassファイルやリソースをまとめる形式です。Goldや実務では重要ですが、最初は存在を知る程度で構いません。

小さな例

javac Main.java
java Main
jar --help

確認ポイント

  • JDKは開発用
  • javacはコンパイル
  • javaは実行
