数値型と文字型

整数型には`byte`、`short`、`int`、`long`があります。小数型には`float`、`double`があります。文字型は`char`です。

数値昇格

`byte`や`short`同士の演算結果は基本的に`int`になります。

byte a = 1;
byte b = 2;
