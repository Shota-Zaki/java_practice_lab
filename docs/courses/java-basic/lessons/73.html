小さなプログラムを設計する手順

初心者がコードを書く前に考えるべき順番を整理します。

入力を決める

プログラムに何を渡すかを決めます。数字、文字列、配列、ファイルなどがあります。

処理を決める

条件分岐、繰り返し、計算、変換など、必要な処理を日本語で書き出します。

出力を決める

最終的に何を表示・返却・保存するかを決めます。

小さな例

// 入力: int score
// 処理: 80以上なら合格
// 出力: "OK" または "NG"

確認ポイント

  • 入力・処理・出力を分ける
  • いきなりコードを書かない
  • 小さく動かして確認する
