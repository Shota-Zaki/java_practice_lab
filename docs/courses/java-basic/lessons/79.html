Lesson 79 Javaの型システムを大きく見る

型とは、値に対して「何ができるか」を決めるルールです。

基本データ型と参照型代入できるかどうか試験での注意

基本データ型と参照型

Javaの型は大きく基本データ型と参照型に分かれます。

分類例
基本データ型int, long, double, boolean, char
参照型String, 配列, 自作クラス, interface型

代入できるかどうか

代入では、左辺の型に右辺の値を入れられるかを見ます。

long a = 10;      // intからlongへ拡大変換
int b = 10L;      // コンパイルエラー
int c = (int)10L; // 明示的キャスト

試験での注意

byteやshortは計算時にintへ昇格します。byte b = 1; b = b + 1;はコンパイルエラーです。
