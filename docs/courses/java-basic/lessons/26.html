1からわかるJava基礎解説 / Lesson 26

IDEとビルドの入口

実務ではIDEを使うことが多いですが、IDEが裏側で何をしているかを理解するとエラーに強くなります。

IDEがやっていること

IDEはソースコードの編集、コンパイル、実行、補完、エラー表示、デバッグをまとめて支援します。ただし、最終的にはjavacやjava相当の処理が走っています。

プロジェクトとソースフォルダ

Javaプロジェクトではsrcの下にソースコードを置き、ビルド結果はbin、out、targetなどに出力されます。フォルダ構成を理解していないと、パッケージやクラスパスのエラーで詰まります。

初心者の落とし穴

IDEで動くのにコマンドで動かない場合、クラスパス、カレントディレクトリ、パッケージ構成が原因であることが多いです。
