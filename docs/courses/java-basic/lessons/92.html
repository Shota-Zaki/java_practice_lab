Lesson 92 ラムダ式の入口

ラムダ式は、処理そのものを値のように渡すための書き方です。

関数型インタフェース引数と戻り値注意

関数型インタフェース

抽象メソッドが1つだけのinterfaceに対してラムダ式を使えます。

interface Action { void run(); }
Action a = () -> System.out.println("Hi");

引数と戻り値

interface Calc { int add(int a, int b); }
Calc c = (a, b) -> a + b;

注意

ラムダ式単体に型があるのではなく、代入先の関数型インタフェースによって意味が決まります。
