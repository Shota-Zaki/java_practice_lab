クラスパスで起こる混乱

コンパイルできたのに実行できない原因を整理します。

classファイルが見つからない

javaコマンドは実行対象のclassを探します。場所が違うとClassNotFoundException系のエラーになります。

packageと実行位置

package付きクラスは、パッケージ名込みで実行します。com.example.Mainならjava com.example.Mainです。

-cpの役割

-cpはclassファイルやライブラリを探す場所を指定します。

小さな例

javac -d out src/com/example/Main.java
java -cp out com.example.Main

確認ポイント

  • package名込みで実行する
  • -dで出力先を指定できる
  • -cpで検索場所を指定する
