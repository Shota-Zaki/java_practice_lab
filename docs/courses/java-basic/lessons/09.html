Java基礎 Lesson 9

for・while

繰り返しとbreak/continueを読む。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

for文

初期化、条件、更新の順で動く。ループ本体の後に更新式が実行される。

while

条件を先に判定する。最初からfalseなら一度も実行されない。

do-while

本体を実行してから条件判定するため、最低1回は実行される。末尾のセミコロンが必要。

breakとcontinue

breakはループを抜ける。continueは次の反復へ進む。for文ではcontinue後に更新式へ進む。

試験での罠

continueの後にある同一ブロック内の文は到達不能になる場合がある。到達不能コードはコンパイルエラー。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
