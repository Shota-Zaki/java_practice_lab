式と文の違い

Javaのコードを「値を作る部分」と「処理を表す部分」に分けます。

式は値になる

1 + 2、x > 0、method()などは式です。評価されると値になります。

文は処理の単位

代入文、if文、for文、return文などは処理のまとまりです。

試験での重要性

switch式は値を返せますが、switch文は処理です。この違いはJava Silverでよく問われます。

小さな例

int x = 1 + 2;
if (x > 0) {
    System.out.println(x);
}

確認ポイント

  • 式は値になる
  • 文は処理の単位
  • switch文とswitch式を区別する
