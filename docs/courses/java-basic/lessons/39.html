式の評価順を読む

Javaの式が左からどのように評価されるかを理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

評価順の基本

多くの式は左から評価されます。ただし、演算子の優先順位と短絡評価に注意が必要です。

int n = 0;
System.out.println(n++ + "," + ++n); // 0,2

短絡評価

&&は左辺がfalseなら右辺を評価しません。||は左辺がtrueなら右辺を評価しません。

int x = 0;
boolean b = false && ++x > 0;
System.out.println(x); // 0

試験での注意

インクリメント、文字列結合、短絡評価が混ざると間違えやすいです。副作用のある式を見つけてから追うのが安全です。
