Lesson 82 コンパイル時に決まること・実行時に決まること

Java Silverで重要なのは「いつエラーになるか」を分類することです。

コンパイル時に決まること実行時に決まること例

コンパイル時に決まること

  • 変数が宣言されているか
  • 型が合っているか
  • アクセスできるメンバーか
  • checked例外を処理しているか
  • 到達不能コードがないか

実行時に決まること

  • 配列の添字が範囲内か
  • nullに対してメソッドを呼んでいないか
  • キャストが実際の型に合っているか
  • 数値変換やAPI呼び出しで例外が出るか

例

String s = null;
System.out.println(s.length()); // コンパイルは通るが実行時例外
