コンピュータと機械語

プログラムは最終的にはCPUが理解できる命令として実行されます。CPUが直接扱う命令は機械語です。機械語は0と1の並びで表され、人間が直接読み書きするには向きません。

機械語とは

機械語はCPUごとに決まった命令の集まりです。たとえば「メモリから値を読む」「足し算する」「別の場所へジャンプする」といった低レベルの命令を組み合わせて動きます。

高水準言語の役割

JavaやPythonのような言語は、人間が読みやすい形で処理を書けるようにしたものです。ただし、そのままCPUが実行するわけではありません。Javaではソースコードをバイトコードに変換し、JVMが実行します。

Javaで重要な見方

Java Silverでは機械語自体は深く問われません。しかし「ソースコード」「コンパイル」「classファイル」「JVM」の関係を理解していないと、実行コマンドやコンパイルエラーの問題で迷います。
