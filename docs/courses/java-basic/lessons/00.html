Java基礎 準備

始める準備

Javaを始めるために必要なものは、最初は多くありません。JDK、エディタ、作業フォルダ、ターミナルでの確認。この4つを先に固めます。

このページのゴール

Javaのインストール先、コードを書く道具、最初の実行確認を自分で説明できる状態にする。

入れるもの

必要なもの役割おすすめ
JDKjavacでコンパイルし、javaで実行するための開発キットEclipse Temurin JDK または Oracle JDK
エディタ.javaファイルを書く。補完、検索、エラー表示も担当するVS Code + Extension Pack for Java
ターミナルコマンドでコンパイル・実行するWindowsならPowerShellまたはWindows Terminal
作業フォルダ学習用コードを保存する場所C:\java-work のような単純な場所

Javaインストーラーへのリンク

JDKは公式配布元から入れてください。検索結果の広告や非公式ミラーから入れる必要はありません。

Eclipse Temurin JDK

無料で使いやすいOpenJDK配布版。学習用ならまず候補にしてよい。

Oracle Java Downloads

Oracle公式のJDKダウンロードページ。バージョンを確認して入れる。

選び方

学習で混乱したくないなら、使うバージョンを1つに固定します。新しいLTS版を入れる場合でも、サイト内で扱う構文より新しい書き方をむやみに使わないことが重要です。

おすすめのエディタ

候補向いている人注意点
Visual Studio Code + Extension Pack for Java最初の学習、短いコードの確認、軽い環境で進めたい人JDKが入っていないとJavaコードは実行できない。拡張機能だけでは不十分。
IntelliJ IDEA Community Edition補完・リファクタ・プロジェクト管理まで強めに使いたい人最初は機能が多く、画面に振り回されやすい。学習初期は小さなファイルで試す。
Eclipse IDE昔からあるJava IDEで学びたい人、授業や職場で指定されている人ワークスペースやプロジェクトの概念に慣れる必要がある。
このサイトでのおすすめ

最初はVS Codeで十分です。javacとjavaの動きを理解してから、必要に応じてIntelliJ IDEAやEclipseへ移る方が混乱が少ないです。

インストール後の確認

JDKを入れたら、PowerShellまたはターミナルで次を実行します。

java -version
javac -version

両方でバージョンが表示されれば、最低限の準備はできています。

状態意味対応
java -versionだけ通る実行環境だけ見えていて、コンパイラが見えていない可能性JDKを入れたか、PATHが通っているか確認する
javacが認識されないJDK未インストール、またはPATH設定の問題JDKを入れ直すか、インストーラーのPATH設定を確認する
古いバージョンが出る別のJavaが先に見つかっている複数JDKが入っていないか確認する

作業フォルダを作る

最初はパスで詰まらないように、英数字だけの短いフォルダ名にします。

mkdir C:\java-work
cd C:\java-work

日本語や空白を含むフォルダ名でも動くことはありますが、最初の段階では余計な原因を減らす方が合理的です。

最初に動かすコード

C:\java-workにMain.javaを作り、次を保存します。

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

保存したら、同じフォルダで次を実行します。

javac Main.java
java Main

Hello Javaと表示されれば成功です。

ここで覚えること

javacはファイル名を指定します。javaはクラス名を指定します。java Main.classではありません。

JDK / JVM / javac / java の役割

用語役割雑に言うと
JDKJava開発に必要な道具一式開発キット
JVM.classを実行する仮想マシンJavaを動かす実行基盤
javac.javaを.classへ変換するコンパイラ
javaJVMを起動し、指定したクラスのmainを実行する実行コマンド

この違いを曖昧にしたまま進むと、エラーが出たときに原因を切り分けられません。

やらない方がいいこと

非公式サイトからJDKを入れる

開発環境は公式配布元から入れる。余計なインストーラーを踏む必要はありません。

複数のJDKを同時に入れる

最初から複数入れると、どのjavaやjavacを使っているのか分からなくなります。

エディタの実行ボタンだけに頼る

ボタン実行は便利ですが、裏で何が起きているか見えにくいです。最初はjavacとjavaを一度は手で実行してください。

確認問題

次のうち、Javaを始める準備として本当に必要なものはどれか。

  • JDK
  • エディタ
  • 作業フォルダ
  • 有料IDE
  • Webサーバー
考え方と答えを見る

最初に必要なのは、JDK、エディタ、作業フォルダです。有料IDEやWebサーバーは不要です。

Javaの基礎学習では、.javaを書き、javacでコンパイルし、javaで実行できれば十分です。
