Lesson 86 メソッド呼び出しは「型」と「実体」を分けて読む

継承問題では、変数の型と実際のオブジェクトの型を分けて考えます。

オーバーライドフィールドは多態的ではない注意

オーバーライド

A a = new B();
a.method();

インスタンスメソッドがオーバーライドされていれば、実体であるB側が呼ばれます。

フィールドは多態的ではない

class A { int x = 1; }
class B extends A { int x = 2; }
A a = new B();
System.out.println(a.x); // 1

注意

メソッドは実体、フィールドは参照変数の型。この区別はSilverで非常に重要です。
