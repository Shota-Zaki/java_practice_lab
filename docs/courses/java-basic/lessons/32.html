ラムダ式の入口

Java Silverでは深くは扱いませんが、Java Goldへ進むならラムダ式の考え方を早めに知っておくと楽です。

ラムダ式とは

ラムダ式は、処理を値のように渡すための記法です。匿名クラスを短く書く仕組みとして理解すると入りやすいです。

Runnable r = () -> System.out.println("run");
r.run();

関数型インタフェース

抽象メソッドを1つだけ持つインタフェースを関数型インタフェースと呼びます。ラムダ式はその実装として使えます。

試験での注意

Java Silver中心なら優先度は高くありません。ただしGoldではStream APIとセットで重要になります。
