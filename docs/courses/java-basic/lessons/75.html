基礎からBronze・Silverへ進む判断基準

基礎解説を終えた後に、どのコースへ進むべきか整理します。

Bronzeへ進む人

Javaの文法にまだ不安があり、変数・if・for・クラスの基本を反復したい場合はBronzeが向いています。

Silverへ進む人

基本文法を一通り読み、コンパイル可否や出力結果問題に慣れたい場合はSilverへ進みます。

Goldへ進む人

Silver相当の文法を理解し、ラムダ式、Stream、並行処理、I/O、モジュールへ進みたい場合はGoldです。

// 基礎 → Bronze → Silver → Gold
// ただし目的によって途中からでもよい

確認ポイント

  • 基礎に不安があるならBronze
  • 試験対策ならSilver
  • 応用APIならGold
