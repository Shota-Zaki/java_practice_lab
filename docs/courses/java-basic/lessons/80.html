Lesson 80 クラスファイルと実行単位

Javaではソースファイル、クラス、クラスファイル、実行コマンドの関係を整理しておく必要があります。

publicクラスとファイル名複数クラスを書けるか実行できるクラス

publicクラスとファイル名

ソースファイルにpublic class Sampleがある場合、ファイル名はSample.javaでなければなりません。

// Sample.java
public class Sample { }

複数クラスを書けるか

1つのソースファイルに複数のトップレベルクラスを書くことはできます。ただし、publicにできるトップレベルクラスは最大1つです。

実行できるクラス

実行対象にはpublic static void main(String[] args)などのmainメソッドが必要です。コンパイルできることと実行できることは別です。
