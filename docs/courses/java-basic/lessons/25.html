1からわかるJava基礎解説 / Lesson 25

package・import・classpath

パッケージはクラスの住所です。importは別パッケージのクラス名を短く書くための仕組みです。

packageとは

packageはクラスを分類する名前空間です。package ex.sample; と書いたクラスは、基本的にex/sample/のようなフォルダ構造と対応します。

package ex.sample;
public class Item { }

importとは

importはクラスを読み込む命令というより、完全修飾名を短く書くための宣言です。java.langパッケージは自動的に使えます。

import java.util.ArrayList;

ArrayList<String> list = new ArrayList<>();

試験での注意

import ex.*; はexパッケージ直下だけが対象です。ex.aやex.bのようなサブパッケージまでは含みません。static importはstaticメンバーを短く書くためのもので、通常のクラスimportとは別です。
