クラスを分割して読む練習

複数クラスのコードを読むときの基本を整理します。

まずクラスの関係を見る

extends、implements、フィールド、コンストラクタ、メソッドを確認します。

次にmainを見る

実際にどのクラスのインスタンスが作られ、どの型の変数に代入されているかを追います。

参照型と実体を分ける

A a = new B(); のようなコードでは、変数の型はA、実体はBです。呼び出せるメソッドと実行されるメソッドを分けて考えます。

小さな例

class A { void m() { System.out.println("A"); } }
class B extends A { void m() { System.out.println("B"); } }
A a = new B();
a.m();

確認ポイント

  • 変数の型と実体を分ける
  • 呼び出せるかは変数の型で見る
  • オーバーライド時の実行は実体で決まる
