演算子の優先順位をざっくり理解する

すべて暗記するより、試験で効く部分に絞って整理します。

丸括弧が最優先

迷ったら()が最優先です。問題を解くときも、まず括弧内を処理します。

後置++と前置++

num++は値を使った後に増えます。++numは増やしてから値を使います。

文字列連結に注意

+は数値加算にも文字列連結にも使われます。左から評価され、途中でStringになると以降は連結になります。

小さな例

int n = 1;
System.out.println(n++ + "," + ++n);

確認ポイント

  • ()を先に見る
  • ++の前置・後置を区別する
  • Stringが混ざると+は連結になる
