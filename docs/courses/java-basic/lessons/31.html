ジェネリクスの入口

ジェネリクスは、コレクションに入れる型を指定する仕組みです。最初はListの意味から理解します。

なぜ必要か

Listに何でも入れられると、取り出したときに型が分からず危険です。ジェネリクスを使うと、コンパイル時に型のミスを見つけやすくなります。

List<String> names = new ArrayList<>();
names.add("Java");
// names.add(10); // コンパイルエラー

ダイヤモンド演算子

右辺の<>はダイヤモンド演算子です。左辺から型を推論できる場合に使えます。
