Java基礎 Lesson 16

interfaceの入口

implements、抽象メソッド、defaultの入口。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

interface

実装すべき振る舞いを定義する型。クラスはimplementsで実装する。

抽象メソッド

interfaceの抽象メソッドは暗黙にpublic abstract。実装クラスではpublicで実装しなければならない。

defaultメソッド

interface内に実装を持てるメソッド。複数interfaceから同じdefaultメソッドを受け継ぐと衝突する場合がある。

interfaceの変数

interfaceのフィールドは暗黙にpublic static final。代入し直せない。

試験での罠

void run(); を実装するときに void run() {} とアクセス修飾子なしで書くと、publicを弱めるためコンパイルエラー。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
