Lesson 108 例外メッセージを読む

例外が出たら、例外名・メッセージ・行番号を見ます。

見る場所

  • 例外クラス名
  • 原因メッセージ
  • 自分のコードの行番号

例

Exception in thread "main" java.lang.NullPointerException
    at Main.main(Main.java:5)

注意

最初に見るべきは長いスタックトレース全体ではなく、自分が書いたファイル名と行番号です。
