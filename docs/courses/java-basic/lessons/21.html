Java基礎 Lesson 21

ビット・バイト・2進数

必要最小限のコンピュータ基礎。

このレッスンのゴール

用語を暗記するだけではなく、Javaのコードを読んだときに「なぜそうなるか」を説明できる状態にする。

bit

bitは0か1の情報。コンピュータの最小単位。

byte

通常8bitで1byte。文字、数値、ファイルサイズなどの説明で使われる。

2進数

0と1で数を表す。Javaの整数リテラルでは0b1010のように書ける。

16進数

0xFFのように書く。色やビット表現でよく出る。Silverでもリテラルとして問われることがある。

確認ポイント

  • 言葉の定義を自分の言葉で説明できるか。
  • コード例のどこで値や参照が変わるか追えるか。
  • コンパイル前に決まることと、実行中に決まることを分けられるか。
