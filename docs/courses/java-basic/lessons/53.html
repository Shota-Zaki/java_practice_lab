型推論varを初心者向けに整理する

varが型を消す機能ではないことを理解します.

このページの目的：Javaを丸暗記ではなく、動く仕組みから理解するための補助解説です。

varの正体

varはコンパイラが右辺から型を推論するための書き方です。実行時に型が変わるわけではありません。

var x = 10; // int
// x = "A"; // コンパイルエラー

使えない場所

フィールド、メソッドの戻り値型、仮引数型には使えません。

class A {
    // var x = 10; // フィールドではNG
}

試験での注意

var a = null;は型を推論できないためコンパイルエラーです。
