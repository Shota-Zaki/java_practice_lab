1からわかるJava基礎解説

プログラミング未経験者でも読めるように、機械語・JVM・コンパイルから順番に整理します。このコースは解説専用です。

コンピュータと機械語

Javaを学ぶ前提として、CPU、メモリ、機械語、アセンブリ、プログラミング言語の関係を理解します。

OS・JDK・JRE・JVM

OS、JDK、JRE、JVMの役割を整理します。

ソースコードから実行まで

`.java`、`.class`、コンパイル、実行の流れを詳しく説明します。

mainメソッドと起動引数

mainメソッド、String[] args、String... argsを説明します。

変数・型・メモリの考え方

プリミティブ型、参照型、変数、メモリの見方を説明します。

数値型と文字型

byte、short、int、long、float、double、charを説明します。

文字列の基本

Stringの不変性、equals、文字列結合を説明します。

条件分岐

if、else、switch文の基本を説明します。

繰り返し

for、while、do-while、拡張for文を説明します。

配列とコレクションの入口

配列、ArrayList、Listの基本を説明します。

クラスとオブジェクト

クラス、インスタンス、フィールド、メソッドを説明します。

staticの考え方

staticフィールド、staticメソッド、インスタンスメンバーの違いを説明します。

コンストラクタ

コンストラクタ、this()、super()を説明します。

継承の基本

extends、オーバーライド、ポリモーフィズムを説明します。

interfaceの基本

interface、implements、defaultメソッドを説明します。

例外処理の基本

try、catch、finally、throwsを説明します。

コンパイルエラーと実行時例外

コンパイルエラーと実行時例外の違いを説明します。

開発環境とファイル構成

フォルダ、パッケージ、クラスパスの入口を説明します。

デバッグの基本

エラーメッセージ、行番号、変数確認の基本を説明します。

Java学習の進め方

初心者向けの学習順を説明します。
