Java Basic

1からわかるJava基礎解説

Java基礎は、最初に必要な内容へ絞った解説コースです。始める準備、コンピュータの仕組み、JDK/JVM、型、参照、制御構文、クラス、例外を順番に理解します。高度なAPIや発展論点には進まず、コードを読むための土台を固めます。

始める準備から始めるLesson 1へ進むBronzeへ進む学習記事を見る
最初に準備するものJDK、エディタ、作業フォルダ、ターミナル確認。ここを固めてからLesson 1へ進みます。

基礎解説の使い方

各Lessonは、量を削らずに「なぜそうなるか」を説明しています。特に、混同しやすい境界線、コードを読む手順、よくある誤解、答えを言語化する確認問題を入れています。

読み方は単純です。まず既存の説明で全体像を掴み、次に「丁寧な補講」で自分の理解が曖昧な部分を潰してください。説明できない用語が残っているなら、問題演習へ急ぐより該当Lessonを読み直した方が効率的です。

Lesson 1〜24の読み方マップ

基礎コース全体を、単発の用語集ではなく「実行の仕組み」から「コード読解」までつながる流れとして読めるように整理しました。焦って問題演習へ進むより、どのLessonが何の土台になるかを意識した方が理解が残ります。

範囲主題ここで説明できるようにすること
始める準備開発環境JDK、エディタ、作業フォルダ、java -version、javac -version。
Lesson 1〜4Javaが動く入口ソースコード、JDK、JVM、main、コマンドライン引数。
Lesson 5〜7型・値・String変数に何が入るか、値と参照、Stringの不変性。
Lesson 8〜10制御構文・配列条件評価、ループ順序、配列の添字と参照コピー。
Lesson 11〜14メソッド・クラス・static・コンストラクタ値渡し、インスタンス、クラス側とインスタンス側、初期化。
Lesson 15〜18継承・interface・例外・エラー分類宣言型と実体、interface型、例外の流れ、失敗の切り分け。
Lesson 19〜24読解と実行の補強ファイル名とクラス名、読解手順、2進数、Unicode、メモリ、ターミナル。

学び方

  1. 始める準備で、JDK・エディタ・作業フォルダを整える。
  2. Lesson 1〜4で、Javaがどう動くかを掴む。
  3. Lesson 5〜11で、型・値・参照・メソッドを固める。
  4. Lesson 12〜18で、クラス、static、継承、interface、例外を読む。
  5. Lesson 19〜24で、ファイル、読解手順、メモリ、ターミナル操作を補強する。

基礎コースで最終的にできるようにすること

24本を読み終えた時点で目指す状態は、用語を知っていることではありません。短いコードを見て、次の順番で説明できることです。

  1. このコードはコンパイルできるか。
  2. 実行はどのクラスのmainから始まるか。
  3. 変数には値そのものが入っているのか、参照が入っているのか。
  4. 条件分岐・繰り返し・メソッド呼び出しで、制御がどこへ移るか。
  5. 出力、例外、終了状態がどうなるか。

ここまでを自分の言葉で説明できれば、BronzeやJavaの演習に進んだとき、単なる暗記ではなく理由付きで選択肢を切れるようになります。

次に進む目安

Lesson 1〜18を読んで、ローカル変数とフィールドの違い、Stringの不変性、配列の添字、static、コンストラクタ、例外の分類を説明できるなら、BronzeまたはJava17 Silverの通常演習へ進んでよいです。説明できない項目が残るなら、問題数を増やすより該当Lessonを読み直した方が速いです。
