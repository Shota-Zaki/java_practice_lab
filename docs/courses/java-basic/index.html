Java Basic

1からわかるJava基礎解説

Java基礎は、最初に必要な内容へ絞った解説コースです。始める準備、コンピュータの仕組み、JDK/JVM、型、参照、制御構文、クラス、例外を順番に理解します。高度なAPIや発展論点には進まず、コードを読むための土台を固めます。

基礎コースで最終的にできるようにすること

24本を読み終えた時点で目指す状態は、用語を知っていることではありません。短いコードを見て、次の順番で説明できることです。

  1. このコードはコンパイルできるか。
  2. 実行はどのクラスのmainから始まるか。
  3. 変数には値そのものが入っているのか、参照が入っているのか。
  4. 条件分岐・繰り返し・メソッド呼び出しで、制御がどこへ移るか。
  5. 出力、例外、終了状態がどうなるか。

ここまでを自分の言葉で説明できれば、BronzeやJavaの演習に進んだとき、単なる暗記ではなく理由付きで選択肢を切れるようになります。

次に進む目安

Lesson 1〜18を読んで、ローカル変数とフィールドの違い、Stringの不変性、配列の添字、static、コンストラクタ、例外の分類を説明できるなら、BronzeまたはJava17 Silverの通常演習へ進んでよいです。説明できない項目が残るなら、問題数を増やすより該当Lessonを読み直した方が速いです。
