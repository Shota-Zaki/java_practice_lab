演習の質を上げる読み方

Bronzeは簡単そうに見えて、基本用語と構文を雑に読むと崩れます。正答率だけを追わず、どの行で値が変わったか、どの文が成立しないかを説明してください。

文として成立するか

セミコロン、型、変数名、mainメソッド、配列宣言を先に確認。

値の変化を見る

代入、if、for、配列要素の変更を1行ずつ追う。

用語を曖昧にしない

クラス、インスタンス、メソッド、フィールドを混同しない。
