if文と条件式の読み方

boolean式を小さく分解し、true/falseの流れを読む。

条件式は1つずつ評価する

ここでは複雑な条件式より、比較演算子と論理演算子の基本を確実に読むことが重要です。

int score = 70;
if (score >= 60) {
    System.out.println("pass");
}

よくある間違い

  • = は代入、== は比較
  • 文字列比較に == を使わない
  • else は直前の対応する if に結びつく
