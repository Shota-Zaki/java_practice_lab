Bronze：クラスとオブジェクト

クラス、インスタンス、フィールド、メソッドを整理します。

ポータルへ戻るBronzeトップ

クラスは設計図

class Item {
    String name;
    int price;

    void print() {
        System.out.println(name + price);
    }
}

インスタンス生成

Item item = new Item();
item.name = "Book";
item.price = 1000;

参照変数がインスタンスを指している、という感覚はJavaでも重要です。
