配列の入口

配列の長さ、添字、デフォルト値を理解する。

添字は0から始まる

int[] nums = new int[3];
System.out.println(nums[0]); // 0

int配列のデフォルト値は0です。参照型配列ならnullです。

範囲外アクセス

コンパイルは通っても、範囲外アクセスは実行時に ArrayIndexOutOfBoundsException です。
