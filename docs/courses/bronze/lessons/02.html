Bronze：型・変数・演算子

初学者がつまずきやすい型と演算子を重点的に整理します。

ポータルへ戻るBronzeトップ

変数宣言

int age = 20;
String name = "Java";
boolean ok = true;

ローカル変数は使う前に初期化が必要です。フィールドとは扱いが違います。

演算子の基本

算術、比較、論理、代入を分けて考えます。= は代入、== は比較です。

Bronzeでの注意

難しいAPIより、基本文法の読み間違いを減らすことが重要です。
