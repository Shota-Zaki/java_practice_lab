関数型インタフェース

抽象メソッドを1つだけ持つインタフェースです。

@FunctionalInterface
interface Checker {
    boolean test(String s);
}

ラムダ式

Checker c = s -> s.length() > 3;

メソッド参照

String::length のように、既存メソッドを関数として渡す書き方です。
