Java Gold：Stream API

Streamの中間操作、終端操作、遅延評価を整理します。

学習記事上部

Streamの基本

List<String> list = List.of("a", "bb", "ccc");
long count = list.stream()
    .filter(s -> s.length() >= 2)
    .count();

中間操作と終端操作

filter、map、sorted は中間操作、count、collect、forEach は終端操作です。

注意点

Streamは一度終端操作を行うと再利用できません。
