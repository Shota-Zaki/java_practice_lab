Java11 Silver：型・var・演算子

Java11で問われやすい型推論、数値昇格、演算子を確認します。

ポータルへ戻るJava11 Silverトップ
学習記事上部

varはローカル変数のみ

var x = 10;      // OK
var text = "A";  // OK
// var field = 1;  // フィールドでは不可

数値昇格

byte や short の演算結果は原則 int になります。

複合代入

+= は暗黙キャストが絡むため、通常代入と結果が違うことがあります。
