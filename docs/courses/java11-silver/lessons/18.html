ArrayList頻出操作

add・set・remove・get・sizeを整理する。

removeのオーバーロード

ArrayList<Integer> list = new ArrayList<>();
list.add(1);
list.add(2);
list.remove(1); // 添字1を削除

Integerを値として削除したいときは remove(Integer.valueOf(1)) のようにします。
