StringBuilder頻出操作

append・insert・delete・reverseの動きを確認する。

StringBuilderは可変

StringBuilder sb = new StringBuilder("abc");
sb.append("d");
System.out.println(sb); // abcd

Stringと違い、同じオブジェクトの内容が変わります。
