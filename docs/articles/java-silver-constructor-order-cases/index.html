コンストラクタ呼び出し順の頻出パターン

試験対策と実装理解の両方で使えるよう、例を中心に整理します。

結論

子クラスのインスタンス生成でも、親クラスのコンストラクタが先に実行されます。

例

class A { A(){System.out.print("A");} }
class B extends A { B(){System.out.print("B");} }
new B(); // AB

エラー例

class A { A(int x){} }
class B extends A { B(){} } // super()が呼べずエラー
