Java GoldのCollectors入門

試験対策と実装理解の両方で使えるよう、例を中心に整理します。

目的

Streamの結果をListやMapに集めるためにCollectorsを使います。

Listへ集める

List<String> r = list.stream()
  .filter(s -> s.length() > 2)
  .collect(Collectors.toList());

groupingBy

Map<Integer,List<String>> m = list.stream()
  .collect(Collectors.groupingBy(String::length));
