キャストとinstanceofの読み方

参照型キャスト、ClassCastException、instanceofパターンを整理します。

キャストは実体を変えない

(B)a と書いてもオブジェクトがBに変わるわけではありません。

実体が合わないと例外

実体がBでないのにBへキャストするとClassCastExceptionです。

instanceofで確認できる

instanceofは実行時の実体を確認します。Java17ではパターン変数も使えます。

例

Object o = "Java";
if (o instanceof String s) {
    System.out.println(s.length());
}

試験・学習での注意

  • 実行結果を見る前に、まずコンパイルできる形か確認します。
  • 似た構文でも、型・戻り値・アクセス修飾子で結果が変わります。
  • 暗記だけでなく、なぜその結果になるかを言語化してください。
