復習ロードマップ

問題を増やすだけでは伸びません。間違えた問題の理由を分類し、同じ理由で落とさないようにするのが重要です。

1. 1周目

通常演習を1問ずつ解き、解説を読んでください。正解した問題でも、理由が説明できなければ見直し対象です。

2. 2周目

見直しマークを付けた問題だけを重点的に処理します。間違い理由を「参照」「型」「文字列」「制御」「継承」「例外」に分類してください。

3. 模試A/B/C/D

60問・90分で通し、途中で解説を見ない運用にしてください。後で見直すボタンを使い、最後に戻って確認します。

4. スコアより大事なもの

点数より、間違え方を見ます。コンパイルエラーを見落としたのか、実行時例外を見落としたのか、出力追跡で値を飛ばしたのかで対策が違います。
