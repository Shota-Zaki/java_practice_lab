Java Goldのジェネリクス入門

型パラメータ、ワイルドカード、境界の考え方を入口から整理します。

ジェネリクスは型安全の仕組み

List<String>ならStringを扱うリストとしてコンパイル時に検査できます。

? extends T

読み取り中心の上限境界です。TまたはTのサブタイプを扱います。

? super T

書き込み中心の下限境界です。TまたはTのスーパータイプを扱います。

例

List<? extends Number> nums;
List<? super Integer> ints;

試験・学習での注意

  • 実行結果を見る前に、まずコンパイルできる形か確認します。
  • 似た構文でも、型・戻り値・アクセス修飾子で結果が変わります。
  • 暗記だけでなく、なぜその結果になるかを言語化してください。
