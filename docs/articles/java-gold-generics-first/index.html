ジェネリクス入門

ジェネリクスは、コレクションの中身の型をコンパイル時に守る仕組みです。

List<String>は「Stringだけを入れるList」

ジェネリクスがあることで、取り出した値を安全にStringとして扱えます。逆に、違う型を入れようとするとコンパイルエラーになります。

List<String> list = new ArrayList<>();
list.add("A");
// list.add(10); // コンパイルエラー
String s = list.get(0);

型パラメータは実行時ではなくコンパイル時の安全装置

多くの場合、実行時には型引数の情報は消えます。これを型消去と呼びます。そのため、new T()やT.classのような使い方はできません。
