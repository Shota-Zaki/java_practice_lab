finalの頻出ルール

変数・メソッド・クラスに付くfinalを整理します。

final変数は再代入できない

一度値を入れたfinal変数には再代入できません。

finalメソッドはオーバーライドできない

親クラスのfinalメソッドは子クラスで上書きできません。

finalクラスは継承できない

Stringなどのfinalクラスをextendsすることはできません。

例

final int x = 10;
// x = 20; // 不可

試験・学習での注意

  • 実行結果を見る前に、まずコンパイルできる形か確認します。
  • 似た構文でも、型・戻り値・アクセス修飾子で結果が変わります。
  • 暗記だけでなく、なぜその結果になるかを言語化してください。
