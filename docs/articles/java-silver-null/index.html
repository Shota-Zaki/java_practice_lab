Java SilverのnullとNullPointerException

null代入とメソッド呼び出しの違いを整理します。

nullは参照なし

String s = null;
System.out.println(s); // null
// s.length(); // NullPointerException

配列要素の初期値

String[] a = new String[2];
System.out.println(a[0]); // null
// a[0].concat("x"); // NullPointerException

試験注意

「nullを表示する」のか「nullに対してメソッドを呼ぶ」のかを分けてください。
