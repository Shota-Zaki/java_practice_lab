Javaの実行・宣言ルール（基礎 / Bronze / Java11 Silver / Java17 Silver）

JDK・JVM・javac・javaソースコードから実行までの流れを整理。 コンパイルエラーの原因実行前に止まる理由と直し方。 コード読解の基本手順コンパイル可否、出力、例外を順番に判定。

型・変数・リテラル（基礎 / Bronze / Java11 Silver / Java17 Silver）

値・型・変数・リテラル式と文、プリミティブ型と参照型を区別。 varと型推論varで推論される型と使えない場所。 nullの意味何も参照していない状態とNPEの入口。

String・配列・API（Bronze / Java11 Silver / Java17 Silver）

StringとStringBuilder不変性、戻り値、文字列プール。 配列の初期値と参照添字、初期値、参照コピーを確認。 配列とArrayList固定長と可変長、remove、List.of。 Java API仕様の読み方メソッドの戻り値と例外を読む。

演算子・制御構造（Bronze / Java11 Silver / Java17 Silver）

演算子と制御構造評価順序、短絡評価、if、loopを読む。 if・for・whileの追跡条件、本体、更新の順序を整理。 switch文・switch式・yield文と式、戻り値、yieldを分ける。

クラス・static・record（Java11 Silver / Java17 Silver）

クラス・オブジェクト・インスタンスフィールド、メソッド、staticを整理。 メソッド・引数・戻り値実引数、仮引数、シグネチャを確認。 メソッド呼び出し解決オーバーロード、型変換、null。 recordコンポーネント、アクセサ、コンストラクタ。

継承・interface・override（Java11 Silver / Java17 Silver）

継承・interface・overridesuper、this、is-a、オーバーライドを整理。 アクセス修飾子public、protected、private、パッケージアクセス。 ポリモーフィズムとフィールド隠蔽宣言型、実体、参照を分けて読む。

ポリモーフィズム・キャスト（Java11 Silver / Java17 Silver）

参照の同一性と同値比較==、equals、String、record。 配列の共変性宣言型と実体型のずれを確認。 instanceofパターン変数型判定後に使える変数の範囲。

例外処理（Java11 Silver / Java17 Silver）

throw・throws・catch・伝播例外オブジェクトと制御の流れ。 finallyと例外try、catch、finallyの実行順序。 実行時例外の原因NPE、範囲外、キャスト、parse失敗。

record・sealed・switch式・パターンマッチング（Java17 Silver）

record・sealed・switch式の整理record、sealed、switch式、pattern matching。 sealedとpermitssealed、final、non-sealedの関係。 sealedとrecordの整理混同しやすい宣言ルールを確認。

Stream・Optional・Collectors（Java Gold SE 11 / Java Gold SE 17）

Stream API中間操作、終端操作、遅延評価。 Optionalof、ofNullable、orElse、orElseGet。 CollectorstoMap、groupingBy、joining。 ジェネリクス型安全性、型消去、ワイルドカード。

ラムダ・NIO・並行処理・モジュール（Java Gold SE 11 / Java Gold SE 17）

ラムダ式と関数型インタフェースPredicate、Function、Consumer、Supplier。 PathとFilesPath、Files、normalize、toRealPath。 並行処理ExecutorService、Future、shutdown。 module-info.javarequires、exports、opens、services。

確認・復習（全体）

用語集曖昧な用語を短く確認。 チートシート判定手順をまとめて確認。 直前確認見直し用のチェック項目。
