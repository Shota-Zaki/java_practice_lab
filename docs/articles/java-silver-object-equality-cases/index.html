Java Silverの同一性・同値性で落ちるパターン

試験対策と実装理解の両方で使えるよう、例を中心に整理します。

結論

==は同じオブジェクトか、equals()はクラスごとの同値判定です。

自作クラス

new Item(1).equals(new Item(1))

equals()をオーバーライドしていなければ、基本的に同一性比較と同じです。

record

record Item(int id) {}
new Item(1).equals(new Item(1)); // true

recordはコンポーネントに基づくequals()を自動生成します。
