Java17 Silverの追加論点を正確に読む

Java17 Silverでは、従来のSilver論点に加えて、record、sealed、switch式、instanceofパターン変数を正確に読む必要があります。

recordはフィールド代入クラスではない

record Item(String name, int price) {}
var item = new Item("Pen", 100);
System.out.println(item.name());
// item.name = "Book"; // 不可

recordのコンポーネントはprivate final相当です。アクセサは getName() ではなく name() です。

sealedは許可された型だけが継承できる

sealed class A permits B {}
final class B extends A {}
class C extends A {} // コンパイルエラー

さらに、sealedクラスの直接サブクラスは final、sealed、non-sealed のいずれかで宣言する必要があります。

instanceofパターン変数

Object obj = "Java";
if (obj instanceof String s) {
    System.out.println(s.length());
}

パターン変数は条件が真の範囲でだけ使えます。否定条件や else 側のスコープでよく問われます。
