Java Goldに入る前のラムダ式入門

Silver後にGoldへ進む人向けにラムダ式の考え方を整理します。

ラムダ式とは

処理を値のように渡す書き方です。関数型インタフェースとセットで使います。

Runnable r = () -> System.out.println("run");
r.run();

引数あり

Function<String, Integer> f = s -> s.length();
System.out.println(f.apply("Java"));

注意

ラムダ式はGold範囲で重要です。Silver学習中は「こういう書き方がある」程度で十分です。
