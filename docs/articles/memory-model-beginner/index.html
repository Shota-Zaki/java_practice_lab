参照・メモリ・代入の基本

参照型の代入、メソッド引数、==の混乱を整理します。

参照型変数はオブジェクトそのものではない

参照型変数には、オブジェクトの場所を指す情報が入っていると考えると理解しやすいです。

Sample a = new Sample();
Sample b = a;
System.out.println(a == b); // true

代入すると参照先が変わる

変数に別のnewを代入すると、その変数だけが別のオブジェクトを見るようになります。

Sample a = new Sample();
Sample b = a;
a = new Sample();
System.out.println(a == b); // false

メソッド引数はコピー

参照型をメソッドへ渡しても、変数そのものが渡されるわけではありません。参照値のコピーが渡されます。
