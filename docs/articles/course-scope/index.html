Java基礎・Silver・Goldの範囲を混ぜないための整理

学習サイトで一番まずいのは、基礎学習者にGold論点を混ぜたり、Java17 Silver対策にGold専用APIを混ぜたりすることです。ここでは、どこまでをどのコースで扱うべきかを明確にします。

Java基礎に入れる内容

  • コンピュータ、OS、JDK、JVM
  • javacとjavaの流れ
  • 変数、型、String、配列
  • if、for、while、メソッド
  • クラス、static、コンストラクタ、継承とinterfaceの入口
  • 例外処理の入口、コンパイルエラーと実行時例外の違い

基礎では、コードを読むための最低限に集中します。学習者に余計な分岐を増やさないことが重要です。

Java17 Silverに入れる内容

  • Java SE 17 Silverで問われやすい文法とAPI
  • record、sealed、switch式、instanceofパターン変数
  • String、StringBuilder、ArrayList、配列
  • 継承、interface、例外処理
  • コンパイル可否、実行時例外、出力結果

Silverでは、GoldのStreamや並行処理を混ぜません。Silverは文法・オブジェクト指向・基本APIの判定力が中心です。

Goldに送る内容

  • ラムダ式と関数型インタフェースの詳細
  • Stream API、Collectors、Optional
  • 高度なジェネリクス、ワイルドカード、PECS
  • NIO.2、Date and Time APIの発展
  • 並行処理、ExecutorService、モジュール

これらはGoldでは重要ですが、基礎やJava17 Silverの初期学習には重いです。

判断基準

その内容を知らないとJava17 Silverの問題が解けないならSilverへ入れる。知らなくてもSilver対策が進むならGoldまたは補助記事へ送る。この基準で整理します。
