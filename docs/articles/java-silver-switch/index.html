Java Silverのswitch文とswitch式の違い

フォールスルー、yield、戻り値を整理します。

switch文

int n = 1;
switch (n) {
    case 1: System.out.print("A");
    case 2: System.out.print("B");
} // AB

switch式

String s = switch (n) {
    case 1 -> "A";
    default -> "Z";
};

試験注意

switch式は値を返すため、すべての経路で値が必要です。
