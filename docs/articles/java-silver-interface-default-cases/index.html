interface defaultメソッドの衝突パターン

試験対策と実装理解の両方で使えるよう、例を中心に整理します。

結論

同じシグネチャのdefaultメソッドを複数継承すると、実装クラスで解決が必要です。

例

interface A { default void run(){} }
interface B { default void run(){} }
class C implements A, B {
  public void run(){ A.super.run(); }
}

注意

super.run()ではなく、A.super.run()の形で呼びます。
