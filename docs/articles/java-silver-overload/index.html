Java Silverのオーバーロード解決まとめ

完全一致、拡大変換、ボクシング、可変長引数を整理します。

優先順位

  1. 完全一致
  2. 拡大変換
  3. ボクシング
  4. 可変長引数

例

void test(long x) { System.out.println("long"); }
void test(Integer x) { System.out.println("Integer"); }
test(10); // long

nullの注意

nullは参照型に渡せます。候補が複数あり曖昧ならコンパイルエラーです。
