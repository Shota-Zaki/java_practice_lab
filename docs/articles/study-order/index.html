Java学習のおすすめ順序

基礎、Bronze、Silver、Goldをどう進めるかを整理します。

最初にやること

まずはJava基礎で、コンパイル、main、型、条件分岐、ループ、配列、クラス、例外の入口を押さえます。ここを飛ばすとSilverのひっかけが理解できません。

Bronzeの位置づけ

Bronzeは文法とオブジェクト指向の入口確認です。試験対策というより、Javaの地図を作る段階です。

Silverの位置づけ

Silverは細かい仕様判定です。コンパイル可否、実行時例外、出力結果、参照比較、継承、例外処理を正確に読む必要があります。

Goldの位置づけ

Goldはラムダ、Stream、ジェネリクス、I/O、並行処理、モジュールなど実務寄りです。Silverの土台が浅いと負荷が高くなります。

現実的な進め方

基礎記事を読む、通常演習で1論点ずつ確認する、弱点タグを潰す、最後に模試を解く。この順番が最も無駄が少ないです。
