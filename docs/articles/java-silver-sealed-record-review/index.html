sealedとrecordのJava17 Silver頻出整理

試験対策と実装理解の両方で使えるよう、例を中心に整理します。

sealed

sealedは継承できるクラスを制限します。直接のサブクラスはfinal、sealed、non-sealedのいずれかを指定します。

record

recordはデータ保持用の簡潔なクラスです。アクセサはname()のようにコンポーネント名になります。

注意

recordのフィールドへ直接再代入したり、存在しないgetter名を呼ぶ問題に注意します。
