このサイトについて

Java Practice Labは、Javaの基礎文法から資格試験レベルのコード読解までを、ブラウザ上で反復学習できるようにした学習サイトです。

掲載内容

サイト内の解説、問題文、選択肢、解説は学習用に作成したコンテンツです。特定の企業、団体、試験運営元が提供する公式教材ではありません。

学習機能

通常演習、模試、間違えた問題の復習、見直しマーク、タグ別復習、学習記事を用意しています。学習履歴は利用中のブラウザに保存され、ログインなしで使えます。

商標について

Javaおよび関連する名称は各権利者に帰属します。このサイトは個人学習を支援する非公式サイトであり、各権利者による承認、提携、保証を示すものではありません。
